This directory contains the source code distribution of the PMQTT
(apache pulsar mqtt protocol handler).

General documentation about this version of PMQTT can be found at:
https://github.com/mattisonchao/pmqtt/blob/main/docs/dev.md
In particular, information about building PMQTT from the source
code can be found at:
https://github.com/mattisonchao/pmqtt/blob/main/docs/installation.md

The latest version of this software, and related software, may be
obtained at https://github.com/mattisonchao/pmqtt/releases.  For more information
look at our web site located at https://github.com/mattisonchao/pmqtt/tree/main/docs.
